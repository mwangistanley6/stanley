Merry-Go-Round-App

A modern mobile application designed to manage and streamline group savings and loan activities for members of the Camp Nou savings group. This Android app is built with Jetpack Compose and Firebase, offering a seamless and secure platform for managing group finances.

Key features include secure user authentication, real-time transaction tracking, automated payment reminders, and comprehensive financial reporting. The app implements Material Design 3 guidelines for an intuitive user experience.
